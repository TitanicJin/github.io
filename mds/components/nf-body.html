应用经典布局框架组件

布局框架通过四个插槽，填充相应组件即可完成经典布局

header-slot
skill-slot
main-slot
footer-slot
<template>
    <nf-body :config="bodyConfig" style="height: 600px">
        <div class="box header" slot="iHeader">header-slot</div>
        <div class="box skill" slot="iSkill">skill-slot</div>
        <div class="box main" slot="iMain">main-slot</div>
        <div class="box footer" slot="iFooter">footer-slot</div>
    </nf-body>
</template>
<script>
    export default {
        data() {
          return {
            bodyConfig: {
              skillWidth: 260,
              headerHeight: 50,
              footerHeight: 35,
            }
          }
        },
    }
</script>
<style>
    .box{
        text-align: center;
        width: 100%;
        height: 100%;
    }
    .header{
        background: #34C0F1;
    }
    .skill{
        background-color: #9BADB9;
    }
    .main{
        background: #F2F3F5;
    }
    .footer{
        background: darkgrey;
    }
</style>

参数

参数树 说明 类型 是否必填 默认值
config skillWidth 左侧功能区宽度 Number 是 无
headerHeight 页头高度 Number 是 无
footerHeight 页脚高度 Number 是 无

slot

slot 说明
iHeader 页眉
iSkill 左侧功能区
iMain 右侧主显示区
iFooter 页脚
