应用页眉组件

<template>
    <div style="height: 50px">
        <nf-header :info="info" :config="headerConfig" @clickHeaderMenu="clickHeaderMenu"></nf-header>
    </div>
</template>
<script>
    export default {
        data() {
          return {
            headerConfig: {
              logo: {
                url: "/logo.jpg",
                width: 40,
              },
              title: "Header组件",
              titleIcon: "el-icon-edit",
              menus:[
                {baseId: "example", icon: "el-icon-lock", label: "例子", divided: false}
              ],
            },
            info:"管理员"
          };
        },
        methods:{
            clickHeaderMenu(arg){
                alert(arg.baseId);
            }
        }       
    }
</script>

参数

参数树 说明 类型 是否必填 默认值
config logo url logo图片地址 String 是 无
width logo图片的宽度 String 是 无
title 页眉组件标题 String 是 无
titleIcon 标题icon String 是 无
menus 页眉组件菜单 Array 否 无
info 登录人信息 String 是 无
defaultKey key menus唯一键字段名 String 是 baseId
label menus展示信息字段名 String 是 label
icon menus展示icon字段名 String 是 icon
divided menus下划线字段名 String 是 divided

组件内部能力

功能 说明 触发机制
退出系统 返回登录页并清除store和localStorage 点击退出按钮
菜单功能 执行响应操作 点击页眉菜单项
