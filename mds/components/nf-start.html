# 指南

# 概述

框架集成了Element-UI,因此可以使用Element-UI中的任何组件
框架集成了axios
框架集成了router
框架集成store

# 安装

npm install newlife-components

# 快速上手

本节将介绍如何在项目中初始化 newlife-components

//引入Vue
import Vue from 'vue'
Vue.config.productionTip = false;
//引入Vue初始加载的视图
import App from './App.vue'

//引入框架
import nf from 'newlife-components'
//设置框架的配置.
nf.nfConfig({
  httpConfig: {},
  routers: {},
  stores: {},
  tabs: {},
  apis: {}
});

//将框架载入Vue
Vue.use(nf);

//初始化Vue实例
new Vue({
  el: '#app',
  router: Vue.prototype.$nfRouter,//载入框架的router
  store: Vue.prototype.$nfStore,//载入框架的store
  components: {App},
  template: '<App/>'
});
